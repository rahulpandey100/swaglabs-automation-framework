# Swag Labs Automation Framework

##  Project Description
This project automates the Swag Labs (SauceDemo) web application using Selenium WebDriver with Java and TestNG. It follows a modular Page Object Model (POM) design and supports data-driven testing using JSON.

##  Tech Stack & Environment
- Java: 21 (LTS)
- Selenium WebDriver: 4.21.0
- TestNG: 7.9.0
- Maven: 3.9.2

##  Features Implemented
- Login Automation (Valid & Invalid Scenarios)
- Error Message Validation
- JSON-based Test Data Handling
- Page Object Model (POM) Design Pattern
- Explicit Wait Handling

##  In Progress
- Add to Cart Flow
- Checkout Flow
- Extent Reports Integration
- Screenshot on Failure

##  How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run `testng.xml`
